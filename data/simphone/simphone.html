Simphone —
Physical Computing project

Simphone is the result of the Physical Computing course in Politecnico di Milano, where unused object were reprogrammed thanks to the embedding of an Arduino board, which could interact with the old circuits.
This approach let us transform an old landline into a Simon game.
Project by Alessandro Fagnani, Alice Di Giammartino, Edoardo Guido, Giusy Di Staso.
